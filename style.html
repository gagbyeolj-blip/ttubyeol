windows 11

windows 11는 이전 windows 10를 대체하기 위해 만들어진 os이다

Windows 11은 새로운 디자인과 향상된 생산성 기능을 제공하는 마이크로소프트의 최신 운영체제이다.
스냅 레이아웃, 위젯, 향상된 게임 성능 등을 지원하며 보안 기능도 강화되었다.
